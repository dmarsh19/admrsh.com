Socket Chat

A lightweight AOL Instant Messenger clone.

Sometimes you just need to copy some text to a different machine on a local network. VNC and RDP are a bit overkill for something so simple. And what if you need to go back and forth? And you're running multiple different operating systems? I needed a simple way to pass text back and forth during setups or for testing applications across various machines. Not too bad for less than 300 lines of code!

You can find the entire repo here

Since it's a single script that I distribute to multiple local machines, it's easier to keep the connection config in the script itself. Nothing special being imported and notes for future updates in the docstring.

      179     def report_update_timestamp(self):
      180         """If current timestamp is older than 5 minutes,
      181            update to new time and print to display window.
      182            Called from within display_msg()."""

      184         five_min_ago = datetime.datetime.now() - datetime.timedelta(minutes=5)
      185         if datetime.datetime.strptime(self.timestamp, '%a, %b %d, %Y %H:%M:%S') <= five_min_ago:

      192     def listen_on_pipe(self):

      207 class ChatSocketServer(object):
      208     """Open a socket in server mode, waiting for connections."""
      209     def __init__(self, pipe):
      210         """."""
      211         self.pipe = pipe
      212         self.sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
      213         self.sock.bind((SERVER_HOST, SERVER_PORT))
      214         self.listen()
      215     # END __init__()
      216 
      217     def listen(self):
      218         """When a connection on the open socket is made, start serving."""
      219         self.sock.listen(1)
      220         self.serve()
      221     # END listen()
      222 
      223     def serve(self):
      224         """When a connection is made on the open socket, accept and receive
      225            the message, appending to a single variable for its entirety
      226            (assumes data will always be str). Send the message through the pipe.
      227            Start listening for new connections again."""
      228         msg = ""
      229         conn, addr = self.sock.accept()
      230         while True:
      231             data = conn.recv(1024)
      232             if not data:
      233                 break
      234             msg = msg + data
      235         self.pipe.send_bytes(str(msg))
      236         self.listen()
      237     # END serve()
      238 # END ChatSocketServer

      241 def socket_send_msg(msg=None):
      242     """Send a message through a socket to a host."""
      243     if msg:
      244         sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
      245         sock.connect((CLIENT_HOST, CLIENT_PORT))
      246         sock.sendall(str(msg))
      247         sock.close()
      248 # END socket_send_msg()

      261 def send_and_display_msg(*args, **kwargs):
      262     """Overwrite the callback for gui input Send button. Displays the local message from the
      263        input window to the display window and sends the message through the socket.
      264 
      265        Because this function involves a server process that is separate from the gui, overwriting
      266        outside the gui class is better practice."""

      268     gui.display_local_msg()

      270     try:
      271         socket_send_msg(gui.current_local_msg)
      272     except socket.error:
      273         gui.display_msg('[user is unavailable]\n', ('error',))
